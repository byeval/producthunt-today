## 开始开发

安装依赖

```bash
$ yarn
```

开始构建

```bash
$ yarn dev
```

使用微信小程序开发者工具打开项目目录。

## 发布

```bash
$ yarn build
```

使用微信小程序开发者工具上传版本。
